# Cloud Computing Final Project - Inverted Index and Search

This project implements an inverted index and search system using:

1. Hadoop MapReduce in Java
2. Apache Spark in Python
3. Sequential Python implementation
4. Simple query search script

## Project Goal

Given a collection of text files, build an inverted index mapping each word to the files where it appears and the number of occurrences in each file.

Example:

```text
cloud       doc1.txt:1 doc2.txt:1
computing   doc1.txt:1 doc2.txt:1
